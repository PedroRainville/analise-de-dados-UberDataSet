# analise-de-dados-UberDataSet
Conjunto de dados de viagens da Uber na Índia, com registros de reservas, valores, distâncias, tipos de veículos, localizações de embarque e desembarque, cancelamentos, avaliações de motoristas e clientes, e método de pagamento. Ideal para análises operacionais, financeiras e de satisfação num contexto de mobilidade urbana.
